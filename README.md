# The Trice Dice

**The Trice Dice** — это roguelike 2D-игра, в которой главный герой, живой игральный кубик, попадает в мир азартных игр. В процессе прохождения он сражается с врагами на трёх уникальных уровнях, стилизованных под казино. Данная игра стартовал в рамках университетской дисциплины как проект и курсовая работа. В данный момент разработка продолжается по собственной инициативе.

## Описание проекта
Проект «The Trice Dice» представляет собой 2D-игру с уникальной механикой, в центре которой находится персонаж — живой игральный кубик. Основная идея игры заключается в сочетании динамичных боевых элементов с элементом случайности: при получении урона глобальное состояние героя меняется посредством смены активной способности. Такой механизм смены способности делает каждый игровой сеанс неповторимым и создает глубокую вариативность прохождений.

Игровой процесс организован вокруг серии помещений, соединенных коридорами, где игрок встречает различных противников и собирает оружие. Карта уровня генерируется процедурно при каждом запуске: пространства делятся на комнаты, между ними прокладываются коридоры, и в случайных точках расставляются враги и оружие. Тем самым достигается высокая реиграбельность — даже прохождения одного и того же уровня оказываются разными из-за изменений расположения элементов и порядка встреч с противниками.

Система способностей реализована через менеджер, который хранит набор эффектов: усиление здоровья, увеличение урона, ускорение скорости передвижения, сокращение времени перезарядки, увеличение скорострельности и уменьшение разброса оружия. При каждом ударе по кубику текущая способность случайным образом заменяется новой, при этом предыдущий эффект деактивируется. Игроку демонстрируется название новой способности, что позволяет адаптировать тактику: в одном случае новый эффект может повысить выживаемость, а в другом — сделать передвижение стремительным и агрессивным.

В бою герой использует огнестрельное оружие: пистолет, автомат и дробовик, отличающиеся скорострельностью, уроном за выстрел, точностью и типом стрельбы. Оружие при подборе попадает в инвентарь игрока, а старое сбрасывается в игровой мир. Стрельба реализована появлением пуль — объектов, движущихся по направлению к прицелу. При столкновении с врагом пуля наносит ему урон.

Противники в игре делятся на два базовых типа: ближний бой и дальний бой. Первые пытаются приблизиться к игроку и наносят урон ударами, вторые стреляют из своих орудий издалека.

Дополнительным преимуществом служит эстетика пиксель-арт, выбранная в качестве визуального стиля проекта. Она позволяет создавать лаконичные, но выразительные образы — от самого персонажа (кубика) до врагов и окружения. Интерфейс игры реализован минималистично: отображаются здоровье персонажа, текущая способность и оружие. При смене способности игрок получает визуальное уведомление.

Основу кода составляют модульные скрипты: каждый тип сущности (игрок, враг, оружие, способности, спавнеры) инкапсулирован в свой файл. Взаимодействие между частями строится на Godot-сигналах, что позволяет избежать жесткой связанности. Например, событие гибели врага отправляет сигнал, на который подписан менеджер врагов, уменьшающий счетчик активных противников и при обнулении инициирующий открытие выхода.

«The Trice Dice» сочетает в себе динамичную двухмерную стрельбу, элемент случайных способностей и процедурную генерацию уровней, создавая сбалансированный, но непредсказуемый игровой опыт. Высокая реиграбельность и модульная архитектура кода открывают широкие возможности для расширения проекта: добавления новых способностей, типов оружия, врагов и уникальных событий.

### Основная механика
- Кубик имеет **6 граней**, каждая из которых даёт уникальную способность:
  - Увеличенный урон
  - Повышенное здоровье
  - Ускоренное передвижение
  - Дополнительная броня
  - Усиленная атака
  - Уменьшенный разброс 
- При попадании удара кубик **переворачивается**, случайным образом изменяя свою активную способность.
- Игрок должен использовать эту механику, чтобы адаптироваться к ситуациям и побеждать противников.

## Игровые уровни
1. **Игровые автоматы** — противники в виде анимированных автоматов, стреляющих жетонами.
2. **Покер** — враги-карты с различными эффектами, основанными на мастях и значениях карт.
3. **Рулетка** — уровень с динамическими ловушками и сильным боссом, представляющим собой колесо рулетки.

## Техническая информация
- **Движок**: Godot 4.x
- **Язык программирования**: GDScript
- **Графика**: 2D-растовая
- **Звук и музыка**: собственные и бесплатные ресурсы
- **Поддерживаемые платформы**: Windows
- **Системные требования**:
  - CPU: 2 ядра
  - RAM: 2 ГБ
  - GPU: поддержка OpenGL 3.3 / WebGL 2.0
- **Система ввода**: клавиатура + мышь
- **Сетевой режим**: отсутствует (одиночная игра)
- **Система сборки**: встроенный экспорт Godot

## Установка и запуск для работы с исходным кодом
1. Установите движок Godot по [ссылке](https://github.com/godotengine/godot/releases/download/4.4.1-stable/Godot_v4.4.1-stable_win64.exe.zip)
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/NikTak777/the_trice_dice.git
   ```
3. Откройте проект в Godot.
4. Редактируйте проект и запускайте игру по нажатию кнопки ▶ для просмотра результата!

## Установка и запуск для игроков
1. Скачайте последний релиз The Trice Dice из раздела [Releases](https://github.com/NikTak777/the_trice_dice/releases).
2. Распакуйте в удобное для вас место на компьютере.
3. Запустите файл "The Trice Dice.exe" и наслаждайтесь игрой!
